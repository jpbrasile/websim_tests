Number Addition App

App Guide:

How to Use the App:

  • Enter two numbers in the input fields labeled "Enter first number" and "Enter second number".
  • Click the "Add Numbers" button to perform the calculation.
  • The result will be displayed below the buttons.

Addition Method:

This app uses pure JavaScript to perform calculations:

  • Numbers are added directly in your browser using JavaScript
  • No external API calls are made
  • Results are instant and don't require internet connectivity

URL Parameters:

You can also perform additions by passing numbers in the URL:

  • Add parameters num1 and num2 to the URL
  • Example: index.html?num1=5&num2=3

PowerShell Example:

You can launch the calculator with parameters using PowerShell:

# PowerShell Launch Example
$num1 = 5
$num2 = 3
Start-Process "index.html?num1=$num1&num2=$num2"
